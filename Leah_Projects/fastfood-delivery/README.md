# 🍕 Food Delivery App  

An online food ordering platform where users can browse menus, add items to a cart, and place orders.  

## 🚀 Features  
- View food items and categories  
- Add items to the cart and checkout  
- Secure user authentication  

## 🛠️ Technologies  
- Django  
- Bootstrap  
- JavaScript
